# Nixia Portfolio

![Nixia Portfolio](https://nixia-erp.vercel.app/images/nixia-portfolio.png) 

Sebuah portofolio pribadi yang dirancang dengan antarmuka dan pengalaman pengguna yang menyerupai sistem operasi Linux (Ubuntu). Dibangun dari awal menggunakan React dan Vite untuk menciptakan pengalaman desktop yang interaktif dan imersif langsung di browser Anda.

## ‚ú® Fitur Utama

-   **Antarmuka Desktop**: Lengkap dengan Top Bar, Dock, dan ikon-ikon di desktop.
-   **Sistem Jendela**: Buka beberapa aplikasi dalam jendela yang dapat digeser, diubah ukurannya, diminimalkan, dan dimaksimalkan.
-   **Aplikasi Bawaan**: Termasuk Terminal, Penampil CV, Tentang Saya, dan Pengaturan.
-   **Pemuatan Realistis**: Layar pemuatan yang mensimulasikan progres berdasarkan pemuatan aset gambar yang sebenarnya.
-   **Showcase Proyek**: Menampilkan proyek-proyek lain seperti [Nixia-ERP](https://nixia-erp.vercel.app/) dalam folder interaktif.
-   **Kustomisasi**: Ubah wallpaper desktop melalui aplikasi Pengaturan.

## üöÄ Memulai Proyek

Untuk menjalankan proyek ini secara lokal, ikuti langkah-langkah berikut:

1.  **Clone Repositori**
    ```bash
    git clone https://github.com/feunixs/nixia-portfolio.git
    cd nixia-portfolio
    ```

2.  **Instal Dependensi**
    ```bash
    npm install
    ```

3.  **Jalankan Server Pengembangan**
    ```bash
    npm run dev
    ```

    Aplikasi akan berjalan di `http://localhost:5173`.
